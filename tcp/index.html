The Concert Project


The Concert Project is a 2D side scrolling point and click game that aims to play exactly like Snoopy Concert.

What it will have when it's finished:

  • - SMBX-like Softmodding capabilities that let you modify every part of the game.
  • - Lua scripting and API
  • - The ability to make your own games within it (called "Concerts")
